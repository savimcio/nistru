# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

The full text is at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

## In short

- Be welcoming, patient, and respectful.
- Assume good faith; critique ideas, not people.
- No harassment, discrimination, personal attacks, or sustained disruption.
- Keep discussion focused; public spaces are not therapy or soapboxes.

## Reporting

Report Code of Conduct concerns privately to the maintainer [**@savimcio**](https://github.com/savimcio) — open a [private security advisory](https://github.com/savimcio/nistru/security/advisories/new) (which is maintainer-only visibility) with subject prefix `[CoC]`. Reports are read by the maintainer(s) only. We'll respond within 72 hours and act on confirmed violations with anything from a private warning up to a permanent ban, proportional to the harm.

Please do not report CoC concerns through public issues or pull requests.

## Scope

This applies to all project spaces — the repository, issue tracker, pull-request discussions, release notes, and any associated chat — and to representing the project in public (conferences, social media, talks).
